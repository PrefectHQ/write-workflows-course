Resources

Let's check out a few resources to assist you in the course and beyond.

Slack

🤝 Join the Prefect Slack community to ask questions and get help from the community. Be sure to join the #prefect-fundamentals-course channel.

🤖 The Slack workpsace also contains Marvin, our helpful bot trained on Prefect's docs and codebase. Ask Marvin a question with @marvin, but don't be alarmed if Marvin's has a little attitude.

Prefect docs

📜 The Prefect docs are a great resource for learning more about Prefect. You can even use the AI search feature to ask questions.

Prefect GitHub repository

💻 The Prefect GitHub repository contains the source code for the Prefect client. See the source code, file bug report and feature request issues, or contribute to the project. And don't forget to star the repo!

Course itHub repository

⌨️ The Course GitHub repository contains helpful code as you work through the course.
